Introduction

The Flexible plugin provides a unified approach to the simulation of deformable solids. It uses a three-level kinematic structure: control nodes, deformation gradients, strain measures, and new mappings between these.
This plugin is documented in the the code documentation and in the following pdf: Flexible.pdf
To insert it in the SOFA development project, check the corresponding option in the cmake configuration.

Project structure

  • the C++ code is tplit in different subdirectories.
  • the test suite is in Flexible_test/
  • the documentation starts in doc/index.html and it is linked to the code documentation. The code documentation is generated using command doxygen Doxyfile in directory doc/.
  • the examples are in examples/ . The first one is Flexible.scn, and when runSOFA opens it, an html page pops up, with a link to the plugin documentation.
